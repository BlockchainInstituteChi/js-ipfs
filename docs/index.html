    What is IPFS?
    
            IPFS is a protocol that allows users to collaborate to store files on personal computers around the world. All records are stored using a unique hash signature, and can be uploaded and downloaded from the network of computers, or swarm by anyone, anywhere. Try it for yourself in this window by uploading a file. Once the file is uploaded, copy the 'LINK' attribute and send it to friends so they can try. 
            With IPFS, your files are never stored on a central server, and no one can ever take them away from you. Learn how to use IPFS for free at the Blockchain Institute.
